# ExcelIA-Linda-Torrado
Proyecto de análisis excel IA
Tema del proyecto:
Análisis y predicción de ventas mensuales utilizando herramientas de inteligencia artificial en Excel.
Pregunta clave:
¿Cuál es la tendencia futura de las ventas según el modelo predictivo de Excel?
Fuente de datos:
Datos simulados con base en tendencias de ventas minoristas reportadas por Microsoft Learn y conjuntos de datos abiertos de retail disponibles en Kaggle.
Herramientas de IA utilizadas:
•	Analyze Data (Ideas)
•	Forecast Sheet (pronóstico automático)
Descripción del análisis:
Se organizó una serie temporal de ventas mensuales y se aplicaron las herramientas de inteligencia artificial integradas en Excel para identificar patrones y generar una proyección futura de las ventas.
Conclusiones:
El análisis muestra una tendencia creciente en las ventas históricas. El modelo predictivo indica que las ventas continuarán aumentando en los próximos meses. Aunque existe un margen de incertidumbre representado por los intervalos de confianza, la proyección general sugiere un crecimiento sostenido.
